CHIRPS
get_target_date_for_download

    Calcula el año y mes del archivo CHIRPS a descargar.
    El objetivo es siempre el mes anterior a la 'current_date' proporcionada
    (o la fecha actual del sistema si no se especifica).

    Args:
        current_date (datetime.date, optional): La fecha de referencia para determinar el mes.
                                                Por defecto, usa la fecha actual del sistema.

    Returns:
        tuple[int, str]: Una tupla que contiene el año (int) y el mes (str, con dos dígitos, ej. "01", "12").

extract_chirps_data
    
    Descarga el archivo CHIRPS más reciente disponible y lo descomprime.
    El mes objetivo para la descarga es el mes anterior a la 'current_date' proporcionada.

    Args:
        config (dict): El diccionario de configuración cargado de config.yaml.
        current_date (datetime.date, optional): La fecha de referencia para determinar el mes a descargar.
                                                Si es None, se usa datetime.date.today().

    Returns:
        Path or None: La ruta al archivo TIF descargado y descomprimido, o None si la descarga falla.
    
NOAA
update_oni_data
    
    Downloads the latest ONI data from NOAA, compares it with the local raw CSV,
    appends any new records, and then saves a processed ONI CSV for the model.

    from raw anomaly data, and saves the result.

    Args:
        config (dict): The configuration dictionary, containing 'project_root' and

    Returns:
        Optional[Path]: Path to the processed ONI index CSV file if successful,

compute_average_raster    

    Computes a historical average raster (simple or weighted) for a given month and year.

    Args:
        config (dict): The configuration dictionary for the specific average calculation.
                       This dict should already contain 'project_root' and relevant 'average_calculation' and 'processing' keys.
        current_year_to_process (int): The year for which the average is being calculated (e.g., 2025).
                                       The historical data will be from `current_year_to_process - num_years_for_average` to `current_year_to_process - 1`.
        current_month_to_process (str): The month (MM format, e.g., "06") for which the average is calculated.

    Returns:
        Path or None: The path to the saved average raster file, or None if calculation fails.

prepare_data_source_config
    
    Prepares a specific configuration dictionary for a data source,
    merging the base configuration with the data source's specific configuration.
    This function creates a *copy* of the base_config and adds/overwrites keys
    relevant for the specific average calculation step.

    Args:
        base_config (dict): The full application configuration from YAML.
        data_source_key (str): The key for the specific data source under 'data_sources'
                               (e.g., 'chirps_historical_average' or 'average_for_model_input').
    Returns:
        dict: A new configuration dictionary prepared for the specific data source.
